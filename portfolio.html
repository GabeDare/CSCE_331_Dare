Software Engineer Intern (May 2024 - Aug 2024)

I worked as a Software Engineering Intern at Raytheon Technologies on the Electro-Optical Distributed Aperture System (EODAS) Project. I developed a program to test the accuracy of software actions and assist in future development. When requested by the customer, I ensured that the program's code met the project requirements by performing static code analysis across the entire codebase for our project group. Through this analysis, I identified and resolved an issue with the code output, which uncovered several previously hidden issues. The debugging I performed improved the overall stability and usability of the project's code. Additionally, when the customer raised concerns about the software's operability, I drafted a report explaining why the program functioned as it did and how changes could impact other areas of the project. I also aligned input data between different software development teams to ensure proper functionality.
